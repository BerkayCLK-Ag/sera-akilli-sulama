## 🌱 Ana Proje

### [Akıllı Sera IoT — Sera v3](https://github.com/BerkayCLK-Ag/sera-akilli-sulama)

ESP32 tabanli akilli sulama prototipi:

- 🌡️ DHT11 ortam sicakligi / nemi
- 💧 Toprak nem + su deposu seviyesi
- ⚡ Röle kontrollu dalgic pompa
- 📺 OLED lokal ekran
- 🖥️ FastAPI + canli web dashboard
- 📡 WiFi + UDP otomatik PC kesfi

> Ziraat muhendisligi perspektifiyle sensor kalibrasyonu, sulama esigi ve guvenlik kilidi uygulanmistir.
