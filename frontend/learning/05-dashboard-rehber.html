Ana dashboard = 3 dosya

frontend/ ├── index.html ← HTML (kartlar, menü, tablo) ├── css/style.css ← CSS (kurumsal tasarım) └── js/dashboard.js ← JS (API + grafik + pompa)

Ders 1–4'te küçük parçaları öğrendin. Ana panel aynı mantık, sadece daha büyük.
